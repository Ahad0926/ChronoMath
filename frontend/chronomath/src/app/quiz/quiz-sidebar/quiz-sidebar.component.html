Key Formulas

Quadratic Formula

x = (-b ± √(b² - 4ac)) / 2a

Discriminant (Δ = b² - 4ac)

  • Δ > 0 → 2 Real Roots
  • Δ = 0 → 1 Real Root
  • Δ < 0 → Complex Roots

Visual Aids

Graph of y = ax² + bx + c

Completing the Square

x² + bx → (x + b/2)² - (b/2)²

Al-Khwarizmi’s Method

Al-Jabr (Completion):
Add terms to complete a square on one side of the equation.

Al-Muqabala (Balancing):
Subtract equal terms from both sides to simplify.
